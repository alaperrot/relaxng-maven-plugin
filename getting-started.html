Getting Started

This plugin manipulates Relax NG schemas during your Maven build. It uses Trang to translate between .rng (Relax NG XML format), .rnc (Relax NG Compact format) and .xsd (XML Schema) based schemas.

The most basic usage of the plugin is to simply wire it into your POM and run the translate goal:

    <plugins>
        <plugin>
            <groupId>info.alaperrot.maven</groupId>
            <artifactId>relaxng-maven-plugin</artifactId>
            <version>2.0</version>
            <executions>
                <execution>
                    <goals>
                        <goal>translate</goal>
                    </goals>
                </execution>
            </executions>
        </plugin>
        ...
    </plugins>

By default all *.rng and *.rnc files in src/main/rng are translated with Trang into the an XSD: src/main/rng/foo.rnc will be translated into target/generated-sources/xsd/foo.xsd. Read the full documentation on generate-sources:translate for all available options.
