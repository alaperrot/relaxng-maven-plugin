relaxng:trang

Full name:

us.bryon:relaxng-maven-plugin:1.2-SNAPSHOT:trang

Description:

Use trang to translate one or more schemas from one language to another.

Attributes:

  • Requires a Maven 2.0 project to be executed.
  • Binds by default to the lifecycle phase: generate-sources.

Required Parameters

Name Type Since Description
dest File - output directory for translated schemas.
src File - source directory for schema files - all .rnc, .rng files in this directory are translated by trang (.rnc -> .rng and .xsd, .rng -> .rnc and .xsd).

Optional Parameters

Name Type Since Description
translations List - rather than putting schema files into src/main/schema, individual translations can be listed out one at a time in nested translation elements. Example: 
         <translations>
             <translation>
                 <in>src/main/xsd/foo.rnc</in>
                 <out>${project.build.directory}/foo.xsd</out>
             </translation>
         </translations>
