Getting Started

This plugin manipulates RelaxNG schemas during your maven build. It uses Trang to translate between .rng (RelaxNG XML formate), .rnc (RelaxNG Compact Format) and .xsd (XML Schema) based schemas.

The most basic usage of the plugin is to simply wire it into your POM and run the trang goal:

    <plugins>
        <plugin>
            <groupId>us.bryon</groupId>
            <artifactId>relaxng-maven-plugin</artifactId>
            <version>1.2-SNAPSHOT</version>
            <executions>
                <execution>
                    <goals>
                        <goal>trang</goal>
                    </goals>
                </execution>
            </executions>
        </plugin>
        ...
    </plugins>

By default all *.rng and *.rnc files in src/main/schema are translated with trang into the other of those two formats, and into an XSD, (src/main/schema/foo.rnc will be translated into target/trang/foo.rng AND target/trang/foo.xsd). Read the full documentation on relaxng:trang for all available options.
