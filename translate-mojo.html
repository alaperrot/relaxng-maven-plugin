relaxng:translate

Full name:

info.alaperrot.maven:relaxng-maven-plugin:2.0:translate

Description:

Translate Relax NG schemas to XML Schemas.

Attributes:

  • Requires a Maven project to be executed.
  • Binds by default to the lifecycle phase: generate-sources.

Required Parameters

Name Type Since Description
sourceDirectory File 2.0

Source directory for Relax NG schema files.

All Relax NG schema files (.rnc and .rng) in this directory are translated to XML Schema files (.xsd).


Default value is: ${project.build.sourceDirectory}/main/rng.
targetDirectory File 2.0

Output directory for translated XML Schema files.


Default value is: ${project.build.directory}/generated-sources/xsd.

Optional Parameters

Name Type Since Description
sourceEncoding String 2.0

Source schema file encoding.

Useful for Relax NG Compact or Relax NG with missing or incorrect file encoding specification.
sourceExcludes String[] 2.0

Source filename exclude patterns, relative to sourceDirectory.

Defaults to none.
sourceIncludes String[] 2.0

Source filename include patterns, relative to sourceDirectory.

Defaults to "*.rnc", "*.rng".
targetEncoding String 2.0

Target schema file encoding.

Defaults to ${project.build.sourceEncoding} if specified, else to UTF-8.


Default value is: ${project.build.sourceEncoding}.
